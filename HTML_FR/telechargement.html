Téléchargement du jeu vidéo IATTACK

Mode d'emploi

  1. Téléchargez le fichier compressé selon votre système d'exploitation:

  2. Dézippez le dossier contenu dans le fichier compressé à l'emplacement de votre choix
  3. Ouvrez le répertoire IATTACK Compiled Game et double-cliquez sur le fichier IAttatckGame
  4. Savourez votre expérience de jeu!

NB Windows : Si Windows Defender vous bloque l'accès au jeu, il suffit de suivre le rapide tutoriel de 01net suivant et tout sera résolu: Comment autoriser le lancement du jeu?

NB Linux : Pour autoriser l'éxécution du fichier sous Linux, faites un clique-droit sur le fichier IATTACK Compiled Game Linux.x86_64 et cochez 'Allow executing file as program' dans l'onglet Permission.

Téléchargement du jeu vidéo
Socially Challenged Astronauts

Mode d'emploi

  1. Téléchargez le fichier compressé:

  2. Dézippez le dossier contenu dans le fichier compressé à l'emplacement de votre choix
  3. Ouvrez le répertoire Socially_Challenged_Astronauts_Game et double-cliquez sur le fichier Socially_Challenged_Astronauts
  4. Le jeu étant exclusivement en mode coopératif, branchez à l'ordinateur une manette de jeu afin que le second joueur puisse contrôler son personnage.

NB Windows : Si Windows Defender vous bloque l'accès au jeu, il suffit de suivre le rapide tutoriel de 01net suivant et tout sera résolu: Comment autoriser le lancement du jeu?
