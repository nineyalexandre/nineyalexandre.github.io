Developers

Alexandre Niney , Julien Frenette, Nicholas Chartier et Jean-Mathieu Rousseau

Project context

  • Project type : projet de groupe en cours de conception de jeux vidéo suivi à l'ETS.
  • Game type : puzzle / plateformer game exclusively in two-players cooperative mode
  • Softwares used : Visual Studio 2019, Blender, Unreal 4.26.2 and Sourcetree
  • Plateform : Windows
  • Language : C++

Game story

Wally and Windy, two astronauts set off to conquer space with their faithful crew robots. Unfortunately, the trip doesn't go as planned and their ship crashes on a mysterious planet.
In order to be able to leave and repair their transport, our two heroes must find their crew robots scattered on this hostile planet and bring them back to their ship.

Personal tasks on this project

  • Group deliveries management for each iteration.
  • Local cooperative multiplayer implementation
  • Level design: research for assets, set up of the obtacles and materials.
  • Creation of an icy surface and of a function impacting the physics of the players when they come into contact with this surface
  • Design of a game mechanic that allows the player to change the state of liquid water to vapor by heating it. Hence, the player is able to be propelled into the air.
